# Product Context

## Register

product

## Platform

web

## Purpose

LUTify gives photographers a fast way to inspect and apply the curated V-Log Alchemy looks without installing a desktop application or trusting a server with original photographs.

## Users

The primary user has one or more camera RAW files and wants to compare a restrained neutral rendering against a creative look before exporting edit-ready 16-bit TIFF or convenient Quality 95 JPEG files.

## Core experience

The interface is a focused dark editing workspace: a local queue provides context, controls stay near the comparison, and the two images remain the dominant visual content. The app presents one clear primary action per state and keeps privacy, processing assumptions, errors, and progress visible.

## Product principles

- Local means local: photos never leave the browser.
- Comparison comes before decoration.
- Unknown color semantics are disclosed, never guessed.
- Failures are explicit and never produce fake successful files.
- Full-resolution work is sequential to keep memory bounded.

The behavior source of truth is `docs/spec.md` and `docs/ssot/*/spec.md`.
