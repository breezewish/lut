# Browser LUTify Change Specification

## Summary

Replace the Python-first desktop workflow with a private static browser product and reusable Rust computation layer while retaining a native CLI.

## Acceptance

The change is accepted when local batch selection, embedded and processed preview, built-in look selection, EV, side-by-side Base/LUT comparison, selected and batch TIFF export, explicit errors, native CLI, frozen legacy migration evidence, and corrected-v2 numerical tests pass on Linux.

Lens correction, automatic exposure, adapter LUT composition, and unproven LUT output color profiles are outside this change.
