# Web UI Redesign

## Introduction

RAW Alchemy provides a full-screen editing workspace rather than a page-shaped interface. The redesign improves information architecture, theme support, and control density without changing RAW processing behavior.

## Requirements

- Desktop separates local sources, image comparison, adjustments, and output into stable workspace regions.
- The Base and LUT images remain the dominant content.
- Users can switch between persistent Light and Dark modes.
- Input bodies remain compact while actions retain accessible targets.
- Medium and narrow layouts preserve the workflow order and keep export reachable.
- The interface remains flat, borderless, and shadowless.

## Non-goals

- Changing the processing pipeline, LUT set, preview contract, or export format.
- Adding accounts, remote storage, or server processing.
